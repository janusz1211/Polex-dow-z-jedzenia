# Polex-dow-z-jedzenia
Repozytorium na zadanie 3 z projektowania obiektowego

## Setup for Visual Studio

1. Otwórz `PolexDowZJedzenia.sln` w Visual Studio.
2. Zbuduj rozwiązanie.
3. Uruchom projekt `PolexDowZJedzenia`.

Możesz też użyć CLI:

```bash
dotnet restore
dotnet build
dotnet run --project PolexDowZJedzenia/PolexDowZJedzenia.csproj
```
